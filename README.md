# Algorithms-in-graph-theory.
Repositorio de algoritmos de reconocimiento en teoría de gráficas, desarrollado como parte del servicio académico de la Licenciatura en Ciencias de la Computación (FC, UNAM), semestre 2026-2, bajo la dirección del Dr. César Hernández Cruz. Incluye implementaciones de algoritmos para gráficas completas multipartitas, split, chain y co-chain.
